Tags

(Voltar)

A tradução de TAG é marcação, pode ser também, etiqueta. Se pensar na forma que é utilizada para html, faz todo sentido.

Tags são utilizadas para definir a função de determinada ação.

Por exemplo, se eu quiser informar para o html que determinado texto é um título, eu uso uma tag especifica.

Com isso, posso dizer que uso as TAGS como etiquetas para informar ao html o tipo de formatação.

Heading Tag

(Voltar)

Heading Tag é o nome da tag de título e sub-título, conhecida por h1,h2,h3,h4,h5 e h6.

Sempre que preciso informar ao html que aquele texto é um título ou sub-título, utilizo as tags.

Preciso sempre respetar a ordem númerica. No caso do título é usado a tag H1 e para os sub-títulos é usada do H2 em diante.

P - Paragráfos

(Voltar)

A tag P é utilizada para escrever paragráfos e junto com as tags de título, a tag P vai é uma que estará presente quase sempre.

A definição de um paragráfo é:

Um parágrafo é uma divisão de um texto escrito, indicada pela mudança de linha, cuja função é mostrar que as frases aí contidas mantêm maior relação entre si do que com o restante do texto.
